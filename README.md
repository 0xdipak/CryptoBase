# CryptoBase 

Hi, this is react js project. Some key points of project are :

- react js
- Tailwindcss for styling
- responsive
- react-router-dom [v6]
- login/signup with firebase
- context api
- axios
- dark mode
- search coin functionality
- using realtime coin gecko api for fetching data.

# Future Updates :

- and more...
